팀 구성 리스트

팀 구성 안내
팀은 직접 개설하여 팀장으로 신청하거나, 기존 팀에 팀원으로 신청할 수 있습니다.
팀장일 경우
팀 만들기 → 팀원 신청 받기 → 팀원 승인 → 신청 마감일 전에 팀 구성 확정 → 관리자가 최종 승인
팀원일 경우
참여할 팀에 신청 → 대기 상태 → 팀장의 승인 → 팀장이 구성 확정 → 관리자가 최종 승인
유의사항
  1. - 신청한 팀에서 반려된 경우, 반드시 다른 팀에 새롭게 팀원 신청을 해야 합니다.
  2. - 신청 마감일까지 팀 구성이 확정되지 않을 경우, 해당 비교과 프로그램 신청은 자동 취소됩니다.
  3. - 팀이 비공개로 설정된 경우, 팀장이 설정한 비밀번호 입력 시에만 팀원 신청이 가능합니다.
  4. - 팀장은 팀원 신청 내역을 확인하여 승인 또는 반려 처리할 수 있으며, 신청 마감일 전까지 팀 구성을 최종 확정해야 합니다.
